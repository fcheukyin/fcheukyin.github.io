ダウンロードURL

リポジトリクローン

GitHubDesktopをインストール後、起動してログイン

File → Clone repository


GitHub.com → filterにfcheukyin/consult-manage-appと入力
Local path: 任意パス\ワークスペース名\consult-manage-app
Cloneを押下 → 完了

Angular導入

コマンドプロンプトを開いて以下のコマンドを実行
C:/任意パス/ワークスペース/consult-manage-app/angular-app/npm install -g @angular/cli

フロント用のライブラリ導入

コマンドプロンプトを開いて以下のパスに移動
cd C:/任意パス/ワークスペース/consult-manage-app/angular-app/


コマンドプロンプトを開いて以下のコマンドを実行
Angular Material
ng add @angular/material
Mat-table-filter
npm install –-save mat-table-filter
jquery
npm install –-save jquery
FullCalendar
npm install -–save @fullcalendar/angular @fullcalendar/core @fullcalendar/daygrid @fullcalendar/timegrid
MomentJS
npm install –-save moment

バック用のライブラリ導入

コマンドプロンプトを開いて以下のパスに移動
cd C:/任意パス/ワークスペース/consult-manage-app/nodejs-mysql-api/


コマンドプロンプトを開いて以下のコマンドを実行
ExpressJS
npm install –-save express npm install –-save express-jwt npm install –-save express-handlebars
Body-parser
npm install –-save body-parser
Crypto
npm install –-save crypto
Jsonwebtoken
npm install –-save jsonwebtoken
Sequelize
npm install –-save sequelize
Cors
npm install –-save cors
eJS
npm install –-save ejs
Path
npm install –-save path

動作確認

1. コマンドプロンプトに以下のコマンドを実行
cd C:/任意パス/ワークスペース/consult-manage-app/angular-app/ ng serve
2. コマンドプロンプトに以下のコマンドを実行
cd C:/任意パス/ワークスペース/consult-manage-app/nodejs-mysql-api/ node server.js
3.　http://localhost:8080にアクセス
4.　ログイン画面が表示されること
